# Supported MCUs

Cirkit Designer supports simulation of several popular microcontrollers. Select an MCU below to learn about its simulation capabilities, supported peripherals, and getting started guides.

| Platform | Description |
|----------|-------------|
| [Arduino Uno](./arduino-uno) | ATmega328p-based board, the most popular in the Arduino family |
| [Arduino Mega 2560](./arduino-mega) | ATmega2560-based board with 54 digital pins, 16 analog inputs, and 4 serial ports |
| [Raspberry Pi Pico](./raspberry-pi-pico) | RP2040-based board with dual-core ARM Cortex-M0+ and PIO |
| [ESP32-S3](./esp32-s3) | WiFi-enabled dual-core Xtensa microcontroller by Espressif |
